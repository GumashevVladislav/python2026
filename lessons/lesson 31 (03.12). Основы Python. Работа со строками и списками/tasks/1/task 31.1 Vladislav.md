### Задание 31.1. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая нормализует строку с названием игры.

Требования:
- убрать пробелы по краям
- заменить несколько пробелов внутри строки на один
- привести каждое слово к виду с заглавной первой буквой
- вывести количество символов и слов в результате

### Пример входных данных
```
  VLAD   quest   online 
```

### Пример выходных данных
```
Название: Vlad Quest Online
Символов: 19
Слов: 3
```

### Критерии успешного выполнения
- строка очищена от лишних пробелов
- регистр приведен к нужному виду
- длина и количество слов посчитаны корректно

### Подсказки
- используй `strip`, `split`, `' '.join(...)`, `title()`

### Частые ошибки и ограничения
- не модифицируй строку по индексу
- учитывай пустой ввод
