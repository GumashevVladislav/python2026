### Задание 31.3. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая помогает организовать список уровней игры.

Требования:
- прочитать строку с названиями уровней, разделенными `;`
- удалить пробелы по краям каждого названия
- привести все названия к нижнему регистру
- отсортировать уровни по алфавиту и вывести их с нумерацией

### Пример входных данных
```
  Boss; start ; Middle ; bonus ; boss 
```

### Пример выходных данных
```
1. bonus
2. boss
3. middle
4. start
```

### Критерии успешного выполнения
- строка корректно разбита на элементы
- пробелы и дубликаты обработаны
- список отсортирован и выведен с нумерацией

### Подсказки
- используй `split(';')`, `strip()`, `lower()`, `sorted`

### Частые ошибки и ограничения
- по желанию можешь убрать дубликаты перед сортировкой
- не используй внешние библиотеки
