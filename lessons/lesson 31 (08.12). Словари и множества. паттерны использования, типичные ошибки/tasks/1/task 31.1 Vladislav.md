# Задание 31.1. Словари и множества. паттерны использования, типичные ошибки
## Блок урока
Словарь как структура быстрого доступа — Блок показывает, как превращать повторяющиеся данные в словарь и мгновенно находить значения по ключу.

## Условие задачи
Кампус университета ведёт учёт лабораторий и даты последних инструктажей по технике безопасности. Менеджер кампуса проверяет, где нужно повторить инструктаж. Создай словарь safety_briefing, где ключ — код лаборатории, а значение — дату инструктажа. Затем выведи информацию для переменных lab_query и new_lab.

## Пример входных данных
```python
labs = [
    ('chem-lab', '2025-12-01'),
    ('bio-lab', '2025-12-03'),
    ('robo-lab', '2025-12-05'),
]
lab_query = 'bio-lab'
new_lab = 'nano-lab'
```

## Пример выходных данных
```text
bio-lab -> 2025-12-03
nano-lab -> не проводился
```

## Критерии успешного выполнения
- Список пар преобразован в словарь safety_briefing с ключом 'код лаборатории'.
- Для обращения к словарю используется метод get с значением по умолчанию.
- Выведены ответы для lab_query и new_lab в заданном формате.
- Код не падает при запросе отсутствующего ключа.

## Подсказки
- Пройдись по списку labs в цикле и наполняй словарь safety_briefing.
- Используй множественное присваивание: key, value = запись из labs.
- Для отсутствующих данных возвращай строку 'не проводился'.

## Частые ошибки и ограничения
- Не обращайся к словарю через квадратные скобки, если нет проверки на существование ключа.
- Не оставляй ключи кортежами целиком — нужен именно идентификатор.
- Не дублируй вывод: достаточно одной строки на каждый запрос.
