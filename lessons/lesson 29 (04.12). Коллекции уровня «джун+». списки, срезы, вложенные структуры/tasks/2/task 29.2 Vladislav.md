# Задание 29.2. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Индексация и срезы. Тренируем извлечение разных диапазонов элементов, работу с отрицательными индексами и шагом.

## Условие задачи
Ты готовишь расписание выхода постов в телеграм канале:
1. Есть список времен публикации на неделю.
2. Получи первые четыре слота для утренних постов.
3. Возьми последние два слота, чтобы запланировать новости выходного дня.
4. Сформируй срез, который берет элементы с индекса 1 по 5 включительно.
5. Построй подборку каждого второго слота, начиная с нулевого индекса.
6. Выведи результаты с подписями.

## Пример входных данных
```python
slots = ["08:00", "10:30", "12:00", "14:30", "16:00", "18:30", "20:30", "22:00"]
```

## Пример выходных данных
```
Утро: ['08:00', '10:30', '12:00', '14:30']
Выходные: ['20:30', '22:00']
Дневной блок: ['10:30', '12:00', '14:30', '16:00', '18:30']
Чередование: ['08:00', '12:00', '16:00', '20:30']
```

## Критерии успешного выполнения
- Каждое требование выполнено с помощью срезов без циклов
- Использованы положительные и отрицательные индексы
- Исходный список не изменяется
- Каждый вывод сопровождается поясняющей подписью

## Подсказки
- `[:4]` удобно берет начало списка
- Чтобы включить элемент с индекса 5, укажи `stop=6`
- Хвост списка получишь через `[-2:]`
- Шаг `2` возвращает каждый второй элемент

## Частые ошибки и ограничения
- Не используй шаг 0
- Следи, чтобы границы не давали пустые результаты
- Не копируй список ради одной операции. достаточно срезов
- Не превышай длину списка при указании `stop`
