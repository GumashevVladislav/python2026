# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. Учимся создавать списки, обращаться к элементам по индексу и выводить их в удобном формате.

## Условие задачи
Собери список приложений, которые ты обязательно ставишь на новый телефон. Создай базовый набор, добавь рекомендации друзей в конец, выведи размер списка, покажи третий и последний элементы. Заверши нумерованным перечнем, чтобы быстро проверить, всё ли установлено.

## Пример входных данных
```python
starter_apps = ["мессенджер", "банк", "заметки"]
extras = ["музыка", "карты"]
```

## Пример выходных данных
```
Всего приложений: 5
Третье приложение: заметки
Последнее приложение: карты
1. мессенджер
2. банк
3. заметки
4. музыка
5. карты
```

## Критерии успешного выполнения
- Исходный список создан и дополнен методами списка
- Количество элементов вычисляется через `len`
- Индексация используется для доступа к середине и концу списка
- Вывод оформлен как нумерованный список

## Подсказки
- `extend` быстро добавит сразу несколько приложений
- Индексы `2` и `-1` помогут выбрать нужные элементы
- Нумеруй через `enumerate(..., start=1)` для читаемости

## Частые ошибки и ограничения
- Не обращайся к индексу `2`, если список короче трёх элементов
- Следи, чтобы список не перезаписывался строками
- Не выводи список в виде `['a', 'b']`, проходись циклом
- Проверь, что все добавления выполнены до финального вывода
