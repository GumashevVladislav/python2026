# Задание 33.1. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Введение в генераторы коллекций» показывает. как одно выражение может заменить цикл и сделать код короче без потери смысла.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

У тебя есть список `raw_minutes` с длительностью коротких занятий в минутах. Нужно получить список `status_labels`, где каждое значение превращается в строку формата `"Сессия X: короткая"` если длительность меньше 15 минут или `"Сессия X: ок"` если длительность 15 или больше. Нумерация сессий начинается с 1. Используй только одно выражение list comprehension.

## Пример входных данных
```
raw_minutes = [12, 18, 5, 20]
```

## Пример выходных данных
```
['Сессия 1: короткая', 'Сессия 2: ок', 'Сессия 3: короткая', 'Сессия 4: ок']
```

## Критерии успешного выполнения
- Весь список `status_labels` создаётся одним выражением list comprehension.
- Для нумерации используется `enumerate(..., start=1)`.
- Условие задаёт только два типа строк и не меняет исходный список.
- Формат строки строго `"Сессия N: короткая"` или `"Сессия N: ок"`.

## Подсказки
- Вспомни шаблон `[результат for индекс, значение in enumerate(коллекция, start=1)]`.
- Используй тернарный оператор: `"короткая" if минут < 15 else "ок"`.
- Нумерацию подставляй через f-строку `f"Сессия {i}: ..."` для читаемости.

## Частые ошибки и ограничения
- Не вставляй дополнительных циклов или `append`.
- Не переписывай исходный список, работай только с результатом comprehension.
- Следи, чтобы условие было простым. одно условие на одно выражение.
