# Задание 33.4. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Set comprehensions» помогают быстро получить уникальные значения и убрать повторы без длинных проверок в цикле.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

У тебя есть список `raw_tags`, где встречаются повторяющиеся названия тем и пустые строки. Построй множество `clean_tags`, в которое попадут только непустые строки в нижнем регистре. Используй одно выражение set comprehension.

## Пример входных данных
```
raw_tags = ["Python", "", "python", "Comprehension", "DATA", "data"]
```

## Пример выходных данных
```
{'python', 'data', 'comprehension'}
```

## Критерии успешного выполнения
- Множество `clean_tags` создаётся единственным set comprehension.
- Пустые строки и строки из пробелов игнорируются.
- Все элементы в результате приведены к нижнему регистру.
- Исходный список `raw_tags` не меняется.

## Подсказки
- Воспользуйся методом `strip()` чтобы убрать пробелы по краям.
- Проверяй непустоту строки после `strip()`.
- Применяй `lower()` к каждому элементу уже после обрезки пробелов.

## Частые ошибки и ограничения
- Не добавляй дополнительные вызовы `set()` вокруг comprehension.
- Не пиши цикл `for` с `add`. это нарушит требование.
- Следи, чтобы в списках было не больше пяти пунктов.
