# Задание 33.2. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«List comprehensions» тренирует умение быстро преобразовывать коллекции и добавлять простые фильтры без потери читаемости.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

Есть список `lessons`, где каждая запись — пара `(название, длительность_минут)`. Построй новый список строк `short_lessons`, в который попадут только занятия короче 40 минут. Формат строки: `"<название> → <длительность> мин"`. Используй одно выражение list comprehension.

## Пример входных данных
```
lessons = [
    ("Питон с нуля", 35),
    ("Отладка", 50),
    ("Git на практике", 25)
]
```

## Пример выходных данных
```
['Питон с нуля → 35 мин', 'Git на практике → 25 мин']
```

## Критерии успешного выполнения
- Список `short_lessons` формируется одним выражением list comprehension.
- В результат попадают только пары с длительностью меньше 40 минут.
- Формат строки полностью совпадает с образцом.
- Исходный список `lessons` не изменяется и сохраняет порядок.

## Подсказки
- Вспомни шаблон `[выражение for пара in список if условие]`.
- Распаковывай элементы через `for title, minutes in lessons` для читаемости.
- Используй f-строку `f"{title} → {minutes} мин"`.
- Условие фильтрации ставь в конце comprehension после `if`.

## Частые ошибки и ограничения
- Не добавляй отдельный цикл `for` вне comprehension.
- Не пиши сложные условия, хватит одного сравнения с 40.
- Следи за стрелкой и пробелами в строке, чтобы формат не сбился.
