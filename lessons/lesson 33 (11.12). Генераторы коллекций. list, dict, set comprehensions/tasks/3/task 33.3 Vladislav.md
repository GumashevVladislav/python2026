# Задание 33.3. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Dict comprehensions» учит быстро собирать словари из других структур, не теряя читаемости и контроля над ключами.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

У тебя есть список `topics`, в котором каждое значение — словарь с ключами `"name"` и `"minutes"`. Собери новый словарь `topic_lengths` через dict comprehension, где ключ — название темы, а значение — подпись вида `"короткая"`, `"средняя"` или `"длинная"` по правилам:
- меньше 20 минут → `короткая`
- от 20 до 40 включительно → `средняя`
- больше 40 минут → `длинная`

## Пример входных данных
```
topics = [
    {"name": "List comprehension", "minutes": 25},
    {"name": "Dict comprehension", "minutes": 18},
    {"name": "Set comprehension", "minutes": 45}
]
```

## Пример выходных данных
```
{
    'List comprehension': 'средняя',
    'Dict comprehension': 'короткая',
    'Set comprehension': 'длинная'
}
```

## Критерии успешного выполнения
- Словарь `topic_lengths` создаётся одним выражением dict comprehension.
- Условия распределения по категориям реализованы внутри выражения.
- Все исходные темы попадают в словарь, ключи не повторяются.
- Исходный список `topics` не меняется.

## Подсказки
- Проходи по списку словарей так: `for topic in topics` и обращайся к значениям через ключи.
- Используй тернарные конструкции, вложенные друг в друга, чтобы получить нужное слово.
- Проверь границы условий: 20 и 40 относятся к категории «средняя».

## Частые ошибки и ограничения
- Не создавай временный словарь в цикле и затем наполняй. всё должно быть внутри `{... for ...}`.
- Не забывай про русский текст категорий, он чувствителен к регистру.
- Следи, чтобы условия не перекрывали друг друга и не оставляли значение без категории.
